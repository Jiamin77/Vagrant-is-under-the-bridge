# Vagrant-is-under-the-bridge
Individual work, 2020.9-2020.10

This project focus on the shared use of the dark space under the bridge between homeless people and ordinary people.

This project aims to enhance the safety of the site and the needs of the homeless population by reconfiguring the underpass space, in order to increase the vitality, safety (especially during nighttime), hygiene and practicality of the underpass area.
